PIM-TUI is a fast, keyboard-driven terminal UI written in Go that lets you activate Entra ID roles, PIM groups, and Azure RBAC roles — including across Lighthouse tenants — straight from your terminal, with built-in security tier awareness, real-time status tracking, and full audit trail support.
